ServiceNow Case Copilot (Mock + Ollama)

A beginner-friendly local AI Case Copilot that generates:
- Case summary
- Customer update draft
- Next actions
- Clarifying questions

This repo uses mock ServiceNow case JSON files for safe demos (no credentials).

Run:
1) pip install requests
2) ollama pull llama3.2:latest
3) python case_copilot.py
